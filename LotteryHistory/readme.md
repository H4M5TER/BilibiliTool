* 18年8月底续费所有特权能都够领取亲密度低保，舰长为1，提督为5，总督为20。之前只有总督。舰长和提督的推送信息只在当前房间（提督的价格都比小电视高了也不舍得发公告，太抠门了）。

  与风暴共同编号。
